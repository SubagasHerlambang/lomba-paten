JavaScript Display Possibilities

JavaScript dapat "menampilkan" data dengan berbagai cara:
1. Menulis ke elemen HTML, menggunakan innerHTML.
2. Menulis ke dalam output HTML menggunakan document.
3. write().Menulis ke dalam kotak peringatan, menggunakan window.alert().
4. Menulis ke konsol browser, menggunakan console.log().
