Pengantar HTML

- HTML adalah singkatan dari Hyper Text Markup Language, menjadi dasar pembentukan halaman web.

- Sebagai bahasa markup standar, HTML digunakan untuk merinci struktur setiap halaman web.

- Dengan label pada bagian-bagian konten seperti "this is a heading," "this is a paragraph," dan "this is a link," HTML memberikan kerangka yang jelas untuk pembuatan halaman web yang informatif dan terstruktur."
